# proto — 最小ループ プロトタイプ v0

`prototype-spec.md` の実装。**実行時LLMゼロ・完全オフライン・決定論（seed固定）**。

## 実行

```sh
cd proto
# Web版（PWA）をローカルで試す
npm ci && npm run build:web
npx serve web   # 等の静的サーバで web/ を開く

# 対話型CLI（自分で遊ぶ。情緒検証の本番）
node --experimental-strip-types src/cli.ts            # 前回のセーブから再開
node --experimental-strip-types src/cli.ts --new      # 新しい世界で始める
node --experimental-strip-types src/cli.ts --new --seed 42   # seed指定

# 台本デモ（決定論の検証用。seed固定で毎回同じ世界）
node --experimental-strip-types src/demo.ts
```

### CLI の遊び方
- **街**：迷宮へ潜る／酒場で噂を聞く／年代記を読む／セーブして終える
- **迷宮**：さらに潜る／探索する（深蝕が蓄積・深いほど危険）／地上へ戻る。潜行・探索の折々に**ダンジョン環境イベント**（結晶/囁き/祠/歪み…深度帯で発火・採取や祈りの選択）と**化石との遭遇**が起きる
- **化石と遭遇したら**：調べる（状況を掘り下げ→絆/形質/年代記へ還流：4-12 遭-①）／周辺を捜索する（遺品・手がかり→**伏線を残し後続イベントを解錠**：遭-②）／鎮魂する（変質の時計を巻き戻す）／遺品を継ぐ／立ち去る
- **死んだら**：最後の一手を選ぶ（→ 化石の極が決まる）→ 次の世代へ（先代の血縁/弟子/無関係を選択）
- セーブは `save/world.json` に自動。終了してもあなたの世界は堆積し続ける

## Web版（本体）
- **NetHack型のグリッドダンジョン**：小部屋＋1マス幅通路＋行き止まり＋ループの手続き生成（最近傍接続で必ず連結）、`@`のターン制移動（**スワイプで移動・タップで待機**・矢印キー／`.`待機）、視界(FOV)と記憶表示、階段で深度が実体化。**マップは深いほど広く、`@`を追うカメラがスクロール**（「図」＝実マップを 1:1 で俯瞰。**踏破済みの床をタップするとそこまで自動移動**＝敵が見えたら停止）。敵数はマップ面積＋深度でスケール
- **化石はマップ上の実体（†）**：歩み寄ると再発見シーン（鎮魂/継承/立ち去る）が開く
- **宝箱（▭）**：歩み寄ると開封（NetHack風＝空/拾得/異物・副作用/罠の抽選）。深いほど少し増える
- テレグラフ戦闘（移動＝行き先ハイライト／攻撃＝@の赤明滅・確定命中/確定ダメージ・退けば見切り：4-11A）、深蝕と奇癖、死→最後の一手→世代交代（系譜）→年代記
- **成長**：撃破XPでレベルアップ→ステ4種（体/力/理/心）を選んで伸ばす（職業選択なし：4-11F②）
- **深蝕魔法**：「術」ボタンから歪撃／静止の眼／影渡り。**燃料は深蝕**＝強く戦うほど化石の変質が進む（4-11F③）。術には専用SE＋エリア点滅
- **敵の強さ可視化**：記号＝種類／色＝強さの段（tier1〜5：灰白→黄→橙→赤→紫発光）。鎧蟲・怨霊・石鬼など上位種が深いほど混じる（4-11F）
- 見た目＝方向A（発光グリフ）・縦持ち。「図」＝踏破範囲の俯瞰地図、メニュー（≡）＝年代記＋音のミュート切替
- **音**：Web Audio で合成する効果音（移動/打撃/被弾/開封/階段/死/鎮魂）＋たまに鳴る環境音（水滴・遠い風・軋み／深いほど頻繁）。素材ファイル不要・実行時LLMゼロと整合。初回タップで起動、≡ からミュート可（端末に記憶）

## デモが実証していること

- 最小ループ一周：潜行→深蝕→遭遇→死（最後の一手）→化石化→世代交代（系譜）→噂→再発見→干渉→年代記
- **痕跡保証**：どの再発見テキストにも出自（名前/装備/口癖）が必ず残る
- **干渉の意味**：鎮魂した化石は朽ちるだけ、放置した化石は異形化が進む（4-1C の時計リセット）
- **死の一手→極**：「迷宮を呪う」と怨念極の化石になる（4-10B）
- 永続化：セーブ→ロードの完全往復

## デプロイ

`main` にマージするたび、GitHub Actions（`.github/workflows/deploy.yml`）が PWA をビルドして GitHub Pages に自動デプロイする。
初回のみリポジトリの **Settings → Pages → Source を「GitHub Actions」** に設定すること。
iPhone では Safari で開いて「ホーム画面に追加」するとアプリとして遊べる（オフライン動作・セーブは端末内 localStorage）。
